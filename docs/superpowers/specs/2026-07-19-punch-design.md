# punch — 출퇴근 원탭 기록 웹페이지 설계

작성일: 2026-07-19
상태: 승인됨 (v1 범위 확정)

## 한 줄 정의

출근/퇴근 버튼 하나로 시각을 기록하고 월 근무시간을 자동 합산하는, 서버 없는 웹페이지 1장. GitHub Pages로 무료 공개.

## 배경과 범위 결정 이유

처음 아이디어는 "출퇴근 기록 + 일일보고 자동 정리(Gmail/캘린더 연동 + AI 요약)"였다.
브레인스토밍 과정에서 진짜 필요는 **출퇴근 기록 하나**였고, 나머지는 "그것만 하기엔
민망해서" 붙인 것임이 확인됐다. 작은 필요를 큰 프로젝트로 부풀리면 완성하지 못하고
출퇴근 기록조차 못 쓰게 되므로, v1은 출퇴근만 아주 작게 만든다.
뺀 기능은 `docs/BACKLOG.md`에 이유와 함께 별도 기록한다 (이 문서와 합치지 않는다).

핵심 필요 두 가지:

1. **매번 시간 적는 게 귀찮다** → 원탭 기록 (앱이 담당)
2. **찍는 걸 깜빡한다** → 리마인더 (폰 기본 알람이 담당, 앱은 안 만듦)

리마인더를 앱에 넣지 않는 이유: 웹 예약 알림은 푸시 서버가 필요하고(서버리스 구조가
깨짐), iOS/브라우저 정책상 신뢰도가 낮다. 폰 알람은 비용 0에 신뢰도 ~100%.

## 기능 목록 (v1 전부)

| 기능 | 상세 |
|------|------|
| 원탭 기록 | 출근/퇴근 탭 → 현재 시각 저장. 오탭 방지로 탭 후 3초간 "실행취소" 표시 |
| 수정 | 기록 줄 탭 → 시간 직접 수정, 삭제. 깜빡한 날은 과거 날짜에 수동 추가 가능 |
| 월 합계 | 일별 근무시간, 월 총합 자동 계산 |
| 내보내기 | CSV 다운로드 (엑셀 호환) + JSON 백업/복원 (기기 이동용) |

## 데이터 & 기술

- **저장**: 브라우저 localStorage. `{ 날짜, 출근시각, 퇴근시각 }` 배열. 하루 1건 단순 모델
- **기술**: HTML + CSS + 바닐라 JS 파일 1개 (프레임워크 없음 — 이 크기엔 도구가 짐)
- **배포**: GitHub 공개 저장소 `punch` + GitHub Pages
- **이유**: 서버·DB·로그인 없음 → 유지비 0. 데이터가 브라우저 밖으로 안 나가므로
  프라이버시 자동 해결. 누구나 URL로 쓰되 각자 기록은 각자 브라우저에만 존재

알려진 트레이드오프: 기기 간 기록이 분리되고, 브라우저 데이터 삭제 시 기록도 사라진다.
→ 완화책으로 JSON 백업/복원과 CSV 내보내기를 v1에 포함. 동기화는 백로그.

## 엣지 케이스 처리

- **퇴근 안 찍고 다음날 출근** → 어제 기록에 "퇴근 미입력" 표시, 탭해서 채우도록 유도
- **자정 넘긴 야근** → 퇴근 시각이 출근보다 이르면 "다음날로 계산할까요?" 확인
- **출근 두 번 탭** → 이미 출근 상태면 버튼이 [퇴근]이므로 원천 차단

## 검증 방법 (완성 기준)

폰에서 URL 열어 출근 탭 → 퇴근 탭 → 리스트/합계 확인 → 시간 수정 → CSV 열어보기,
이 흐름이 막힘없이 되는 것. 원탭 기록은 3초 안. 브라우저를 껐다 켜도 기록 유지.
